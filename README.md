3. Какие в perl существуют операторы сравнения?

Ответ:		числа	строки	Назначение оператора
			==	eq	равно
			!=	ne	неравно
			<	lt	меньше
			>	gt	больше
			<=	le	меньше либо равно
			>=	ge	больше либо равно
